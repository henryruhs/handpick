Installation
------------

Install on your system:

```
npm install handpick --global
```


Setup
-----

Define unofficial dependencies:

```json
{
	"lintDependencies":
	{
		"@isnotdefined/eslint-config": "5.1.0",
		"eslint": "8.17.0"
	}
}
```


Usage
-----

Run the command:

```
handpick [options]

-V, --version
-C, --config <config>
-T, --target <target>
-F, --filter <filter>
-M, --manager <manager>
-R, --range <range>
-P, --path <path>
-h, --help
```
